# 抖音商城-微服务

## 项目开发环境
```
douyin-mall-microservice
├─ app # 业务开发代码目录
├─ common # 公共工具库
├─ idl # idl目录
├─ db # 数据库初始化
├─ deploy # 项目部署相关
└─ scripts # 项目脚本
```

进入业务代码目录，执行cwgo命令生成代码。
